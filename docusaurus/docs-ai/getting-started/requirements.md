# Requirements

Before getting started with Brizy AI, ensure you have the following basic requirements.

## API Keys

To use Brizy AI features, you'll need these API keys:

### Required APIs

- **AWS S3 Bucket**: For storing assets and configurations
  - Create a bucket in your AWS account
  - Configure CORS settings for web access

- **OpenAI API Key**: For AI-powered text generation and content creation
  - Get your key from [OpenAI Platform](https://platform.openai.com/api-keys)
  - Free tier available for testing

- **Google Places API Key**: For location-based data and search
  - Get your key from [Google Cloud Console](https://console.cloud.google.com/apis/credentials)
  - Free tier includes 100 requests per day

- **Facebook API Key**: For social media integration and user data
  - Get your key from [Facebook Developers](https://developers.facebook.com/docs/facebook-login)
  - Free tier includes 100 requests per day

- **Unsplash API Key**: For high-quality stock images
  - Get your key from [Unsplash Developers](https://unsplash.com/developers)
  - Free tier includes 50 requests per hour

## Quick Start

1. **Set up your environment** with the API key
2. **Start building** with Brizy AI!

Once you have your API keys ready, proceed to the [Usage Guide](./usage.md) to learn how to integrate Brizy AI into your project.
